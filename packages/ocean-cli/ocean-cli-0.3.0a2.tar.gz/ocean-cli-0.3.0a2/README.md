## Initial development setup
For linux/mac
```
$ ./boot.sh
$ source venv/bin/activate
```
For windows
```
> ./boot.bat
> venv\Scripts\activate.bat
```

## Install
Install ocean-cli for production.
```
pip install ocean-cli
```

If you want to install develop mode
```
pip install -e .
or
python setup.py develop
```
